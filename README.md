# Foodify - Online Food Ordering System

Foodify is a web-based online food ordering platform developed to make food ordering and management seamless for both users and administrators.

## 🚀 Features
* **User Panel:** Browse menu items, view prices, and place food orders.
* **Admin Dashboard:** Manage food menus, track user lists, and handle order statuses.
* **Database Driven:** Built-in SQL schema handling order history and connection states.

## 🛠️ Tech Stack
* **Frontend:** HTML, Tailwind CSS
* **Backend:** PHP
* **Database:** MySQL

## 📁 Project Structure
Brief overview of the core files:
* `adminDashboard.php` - Portal for system administrators.
* `menuitem.php` / `menubar.php` - Displays food categories and items.
* `connection.php` - Handles the database connection logic.
* `Online_food_delivery.sql` - Complete database backup schema.
